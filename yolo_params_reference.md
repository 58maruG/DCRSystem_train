# YOLO パラメータ クイックリファレンス

> 対象: Ultralytics YOLO (v8 / v11 系)
> 「このパラメータ何だっけ？」というときに引くための辞書。チューニング方針は `yolo_tuning_guide.md` を参照。

---

## モデル選択

| モデル | 特徴 | 用途 |
|--------|------|------|
| `yolo11n.pt` | 最軽量・最速 | リソースが厳しいとき、動作確認 |
| `yolo11s.pt` | 軽量・バランス型 | **現在使用中** |
| `yolo11m.pt` | 中間 | 精度を上げたいが重すぎるのは困る |
| `yolo11l.pt` | 重い・高精度 | GPUに余裕があるとき |
| `yolo11x.pt` | 最重量・最高精度 | ベンチマーク用 |

---

## 基本パラメータ

### データ・入力

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `data` | — | `data.yaml` のパス。クラス定義・train/val パスを記載したファイル |
| `imgsz` | `640` | 入力画像サイズ（ピクセル）。大きくすると小物体に有利、VRAMも増える |
| `batch` | `16` | バッチサイズ。VRAMに収まる最大値にするのが基本 |
| `fraction` | `1.0` | データセットのうち何割を使うか（`0.5` にすると半分） |
| `cache` | `False` | `"ram"`: 高速だがメモリを食う / `"disk"`: 中間 / `False`: 毎回読む |
| `workers` | `8` | データ読み込みの並列数。CPU性能が低い場合は下げる |

### 学習の打ち切り・停止

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `epochs` | `100` | 最大エポック数。実質は `patience` で決まることが多い |
| `patience` | `50` | val metric が改善しないエポック数が続いたら自動停止 |
| `close_mosaic` | `10` | 最後の N エポックで mosaic 拡張を無効化する。終盤の安定に効く |

### 学習率・最適化

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `optimizer` | `"auto"` | `SGD` / `Adam` / `AdamW` / `NAdam` / `RAdam` / `RMSProp` |
| `lr0` | `0.01` | 初期学習率 |
| `lrf` | `0.01` | 最終学習率の割合（`lr0 × lrf` が最終値） |
| `momentum` | `0.937` | SGD のモメンタム、Adam の β1 |
| `weight_decay` | `0.0005` | 正則化。過学習気味なら増やす |
| `cos_lr` | `False` | コサインスケジューラを使う。終盤の微調整に有効 |
| `warmup_epochs` | `3.0` | 序盤に学習率をゆっくり上げる期間（エポック数） |
| `warmup_momentum` | `0.8` | ウォームアップ中のモメンタム初期値 |
| `nbs` | `64` | 公称バッチサイズ。実バッチとの差を自動スケールして補正 |
| `amp` | `True` | 自動混合精度（FP16）。高速化・省メモリ。不安定なら `False` |

### データ拡張（Augmentation）

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `mosaic` | `1.0` | 4枚合成拡張の適用率。`0.0` で無効（最終 N エポックは `close_mosaic` で自動無効） |
| `mixup` | `0.0` | 2枚の画像をブレンドする拡張。物体検出では控えめに使う |
| `copy_paste` | `0.0` | 他画像からオブジェクトをコピー貼り付け。**少数クラスに有効** |
| `degrees` | `0.0` | ランダム回転の最大角度（±degrees） |
| `translate` | `0.1` | ランダム平行移動（画像サイズに対する割合） |
| `scale` | `0.5` | ランダムスケール変化（±scale） |
| `shear` | `0.0` | せん断変形の最大角度 |
| `perspective` | `0.0` | 透視変換（小さい値のみ推奨、例 `0.0001`） |
| `flipud` | `0.0` | 上下反転の確率 |
| `fliplr` | `0.5` | 左右反転の確率（デフォルト0.5） |
| `bgr` | `0.0` | BGR/RGB チャンネル入れ替えの確率 |
| `hsv_h` | `0.015` | 色相（Hue）のランダム変化幅 |
| `hsv_s` | `0.7` | 彩度（Saturation）のランダム変化幅 |
| `hsv_v` | `0.4` | 明度（Value）のランダム変化幅 |
| `auto_augment` | `"randaugment"` | 自動拡張ポリシー（`"randaugment"` / `"autoaugment"` / `"augmix"` / `""` で無効） |
| `erasing` | `0.4` | 分類タスク用のランダム消去確率 |

### ロス重み

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `box` | `7.5` | バウンディングボックス位置のロス重み |
| `cls` | `0.5` | クラス分類ロスの重み。クラス不均衡があるなら要調整 |
| `dfl` | `1.5` | 分布焦点ロス（Distribution Focal Loss）の重み |

### 保存・出力

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `project` | `"runs/detect"` | 結果の保存先ディレクトリ |
| `name` | `"train"` | 実行名（同名があると `train2`、`train3` と連番になる） |
| `save` | `True` | チェックポイント（`best.pt`, `last.pt`）の保存 |
| `save_period` | `-1` | N エポックごとにチェックポイント保存（`-1` で無効） |
| `plots` | `False` | 学習曲線・混同行列などのグラフを保存するか |
| `exist_ok` | `False` | 既存の `name` フォルダを上書きするか（`False` だと連番） |

### その他

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `device` | `""` | 使用デバイス（`0` = GPU0、`"cpu"` = CPU、`"0,1"` = マルチGPU） |
| `seed` | `0` | 乱数シード。再現性が必要なときに固定 |
| `deterministic` | `True` | 完全再現モード（やや遅くなる） |
| `pretrained` | `True` | 事前学習済み重みから始めるか（`False` でスクラッチ） |
| `resume` | `False` | 中断した学習を再開するか（`True` にすると `last.pt` から継続） |
| `freeze` | `None` | 凍結するレイヤー番号またはリスト（転移学習で backbone を固定するとき） |
| `dropout` | `0.0` | Dropout 率。過学習対策（検出タスクでは基本0） |
| `single_cls` | `False` | 全クラスを1クラスとして扱う（デバッグ用） |
| `rect` | `False` | 矩形学習（アスペクト比をそろえてパディングを減らす） |
| `multi_scale` | `False` | バッチごとに `imgsz × (0.5〜1.5)` でランダムスケール学習 |
| `val` | `True` | エポックごとに validation を実行するか |
| `verbose` | `True` | ログの詳細出力 |

## よく使う組み合わせメモ

```python
# 少数クラス対策セット
copy_paste=0.3,
mosaic=1.0,
close_mosaic=15,
cls=1.0,       # クラスロスを重くする

# 過学習対策セット
weight_decay=0.001,
dropout=0.1,
augmentation を強化（scale, degrees など）

# 小物体対策セット
imgsz=1280,
multi_scale=True,
scale=0.7,
```
