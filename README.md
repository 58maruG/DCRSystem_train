# サクランボ被害果除去システム_弐号機 学習用のフォルダ

YOLOv8（v11） を用いて、サクランボ被害果の検出モデルを学習・評価するためのリポジトリ。  
学習データの準備から、モデルの学習・予測・評価までを一連で行える。

## フォルダ構成

```
DCRSystem_train/
├── train.py              # 学習スクリプト
├── predict.py            # 推論スクリプト
├── test.py               # 評価スクリプト
├── reborn.py             # 再学習・補助スクリプト
├── shuffle_TrainVal.py   # 学習/検証データ分割スクリプト
└── yolo_SusHi.yaml       # データセット設定ファイル
```

## 環境
- Windows 11
- Python 3.11

## 使用するパッケージ
- "pynput>=1.8.1"
- "ultralytics>=8.3.217"

## インストール
```bash
# ターミナルで以下を実行
cd C:\Users\保存したい場所
git clone https://github.com/58maruG/DCRSystem_train.git
```
初回実行時にクローンしたフォルダに移動し、以下コマンドをターミナルで実行
```bash
uv run python first.py
```
「実行環境」「使用するパッケージ」が自動でインストールされる

## ⚠️ データファイルの取り扱いについて

**このリポジトリは学習データ（画像・アノテーション）を Git 管理しない。**

新しいデータセット（画像 `.jpg`/`.png` やラベル `.txt` など）をこのフォルダに追加した場合、  
**必ず `.gitignore` にそのフォルダ名またはファイルパターンを追記してからコミットすること。**

### `.gitignore` への追記例

```
# 新しいデータセットフォルダを追加した場合
dataset_新しいフォルダ名/

# 特定の拡張子をまとめて除外したい場合
*.jpg
*.png
*.txt  # ← ラベルファイルも除外対象
```

追記せずにコミットすると、大量の画像ファイルがリモートリポジトリに混入するため注意。
